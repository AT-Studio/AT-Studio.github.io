Accessbility

Images

All images featured on the website have an alternative text associated with them so that users that are visually impaired or that are using text-only browsers have the opportunity to understand the information provided by the images.

Site Navigation

For easy and intuitive site navigation, a navigation bar is provided at the top of the screen. Within the navigation page, the text linking to the current page is in bold. This allows users with colorblindness to clearly identify which page they are currently located on. Additionally, hovering over any of the other text links in the navigation bar makes them appear bold, giving the user a visual cue that these items are indeed links.

Links

All of the links within the content (below the navigation bar) of the page automatically become underlines when hovered over. This allows users with colorblindness to easily distinguish regular text from links. Furthermore, links that navigate within the website are opened within the same the same tab, allowing for easy back navigation. However, links to external sites or documents, are opened within a new tab, allowing for an intuitive browsing experience for the user.

Screen Size

To allow for a pleasant browsing experience on multiple screen sizes and devices (desktop, tablet, mobile) this website is being created with an emphasis on responsive design. By utilizing responsive design, users may resize their browsers to any size without sacrificing information and the clean and rich design of the website. Though responsive design has only been fully implemented on the ‘About Me’ and ‘Accessibility’ pages, this feature will ultimately be available on all pages of the website as it nears completion.
